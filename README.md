

Full Stack Multi vendor ecommerce project
Are you looking to build a professional eCommerce website that not only enhances your coding skills but also boosts your resume? This source code is perfect for students, developers, and business owners who want to create a fully functional eCommerce platform using modern technologies like Spring Boot and React. Whether you're looking to learn or create a live website for your business, this project will get you there!
Technology Stack
Backend: Spring Boot, MySQL, Spring Security, Java Mail Sender, JWT
Frontend: React, TypeScript, Redux Toolkit, MUI, Tailwind CSS, React Chart, Formik, Yup, React Router DOM, Axios
Payment Gateways: Razorpay (for Indian students) & Stripe (for international students)
